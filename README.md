# 🌿 Verde – Vegetarian Recipe App

A React app to search and explore vegetarian recipes using the Spoonacular API.

## Stack

- React 18 + React Router v6
- Axios
- Context API + useReducer
- Spoonacular API

## Getting Started

1. Clone the repo
2. Run `npm install`
3. Create `.env` with `REACT_APP_SPOONACULAR_API_KEY=your_key`
4. Run `npm start`

## Features

- 🔍 Search vegetarian recipes
- 🏠 Homepage with random picks
- 📖 Detail page with ingredients & instructions
- 📱 Fully responsive
